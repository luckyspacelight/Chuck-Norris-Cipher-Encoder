Rewriting code using switch

Here is a piece of code that uses the conditional statement if-else-if.

if (val == 1) { 
    System.out.println("ONE"); 
} else if (val == 2) { 
    System.out.println("TWO"); 
} else if (val == 3){ 
    System.out.println("THREE"); 
} else { 
    System.out.println("UNKNOWN NUMBER");
}

How can you rewrite this code using a switch statement? Check all possible options.

a)

switch (val) { 
    case 1: System.out.println("ONE"); 
    case 2: System.out.println("TWO"); 
    case 3: System.out.println("THREE"); 
    default: System.out.println("UNKNOWN NUMBER");
}

b)

switch (val) { 
    case 1: System.out.println("ONE");
       break; 
    case 2: System.out.println("TWO");
       break; 
    case 3: System.out.println("THREE");
       break; 
}

c)

switch (val) { 
    case 1: System.out.println("ONE");
       break; 
    case 2: System.out.println("TWO");
       break;
    case 3: System.out.println("THREE");
       break; 
    default: System.out.println("UNKNOWN NUMBER"); 
}

d)

switch (val) { 
   case 1: System.out.println("ONE");
      break; 
   case 2: System.out.println("TWO");
      break; 
   case 3: System.out.println("THREE");
      break; 
   case 4: System.out.println("UNKNOWN NUMBER");
      break; 
   default: System.out.println("UNKNOWN NUMBER"); 
}
